Wat zijn entiteiten in Java? Een entiteit vertegenwoordigt een object in de database en lijkt op iets zoals je het in de echte wereld ook hebt, zoals gebruikers, producten, adressen, bestellingen, hotelkamers, etc. Deze vertalen we dan naar entiteiten met specifieke eigenschappen, zoals een hotelkamer bijvoorbeeld altijd een kamernummer, aantal slaapplaatsen, of het een roken/niet-roken kamer is en een prijs per nacht. Dit kun je dan als volgt opschrijven: Room(roomNumber, capacity, smokingAllowed, pricePerNightPerPerson)

Wij faciliteren de Novi Educational Backend. dit is een database waar je gebruikers kan opslaan. Meer informatie staat op de website. Je mag ook een database gebruiken van eerdere leerlijn of een firebase gebruikers database. Deze zijn lastiger om te configureren en je krijgt er niet een extra punt voor.

Op Github staat een lijst met API's die je mag gebruiken. Wil je een andere API gebruiken dan mag dat ook. Het is van belang dat je de API naam + URL hier inlever. zodat wij kunnen kijken of de api toegankelijk genoeg is om te gebruiken.
